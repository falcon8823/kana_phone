# KanaPhone

ひらがなを音素列に変換します。

## Installation

Add this line to your application's Gemfile:

    gem 'kana-phone'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install .

## Usage

```bash
$ bin/kana2phone
```
